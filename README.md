
## ✨ 简介

🎯 **CursorPro** 是一款 **完全免费** 的工具，专为自动获取新账号、一键重置新额度、解决机器码问题而设计，支持无限续杯，极简使用体验！

🛠️ 工具亮点：

- 🔓 无需注册使用 Claude 3.7  
- ⚡ 一键获取新额度  
- 📦 自动满额度账号分配  
- 💻️ 无需登录自己账号  
- 🔁 顺带自动解封机器码  

---

## ⚠️ 注意事项

🚨 **使用前，请先随便用一个账号登录一次 Cursor 软件**，确保系统初始化正常，否则部分功能可能无法正常运行。
